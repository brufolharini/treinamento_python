Scripts em desenvolvimento:

No primeiro código (app.py) temos um joguinho de teste como parte de um desafio que tive no curso de Python que estou fazendo;
No segundo código (projeto1.py) criei um executavel onde você pode selecionar a pasta de origem, após pegar e selecionar a
pasta de destino. Após realizar as devidas seleções de pastas, precisa apenas clicar no botão de Organizar Arquivos. E taram
ele irá pegar todos os tipos de arquivos da pasta origem e irá transeferir para a pasta destino separando os arquivos por tipos
cada um em sua respectiva pasta de extensão.
